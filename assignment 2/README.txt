Prerequisites:
Atom Text Editor or Netbeans Java IDE recommended for the program.

Installation:
Unzip the folder, and place in any directory of choosing.

Deployment:
Server.java and ClientConnectionHandler.java must be compiled at the same time
using command line.
Client1.java when compiled will recieve two arguments; Computer Name, and Directory Path. Compile
Client1.java with the following arguments in the command line.
